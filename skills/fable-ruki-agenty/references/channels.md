# Внешние каналы — механика

Дефолт конвейера — штатные субагенты (§ Субагенты и каналы SKILL.md); этот файл нужен, только если на старте конвейера пользователь подключил внешний канал. Здесь всё, что привязано к конкретным продуктам, — текст SKILL.md от канала не зависит.

## Codex (OpenAI)

Роли: **ревьюер финального ревью** — главная ценность, другое семейство моделей снимает модельно-коррелированные слепые зоны; **исполнитель** — по решению пользователя. Модель — сильная рабочая (`gpt-5.5` или актуальная ей на смену), всегда явным флагом, не из конфига. Бонус канала: жжёт подписку OpenAI, а не лимиты Anthropic.

### Подключение — проверить в этом порядке

1. **Плагин Claude Code** — в списке типов агентов есть `codex:codex-rescue`. Прекондишн (прогоняет субагент, не оркестратор — запрет № 1):

```bash
CODEX_PLUGIN="$(printf '%s\n' "$HOME"/.claude/plugins/cache/openai-codex/codex/*/ | sort -V | tail -1)"
node "${CODEX_PLUGIN%/}/scripts/codex-companion.mjs" setup --json   # ждать ready:true и auth.loggedIn:true
```

Не ready — пользователю одна строка: нужен `/codex:setup` (установка и логин — его решение); роли остаются субагентам.

2. **CLI без плагина** — в системе есть `codex` (проверка субагентом: `codex --version`). Вызовы — Bash-субагентом в headless-режиме: `codex exec "<конверт>"`. Механика в этом конвейере не обкатана: первый прогон — под наблюдением, результат в журнал сбоев.

### Операции (плагин)

- **Диспатч** — Agent tool `codex:codex-rescue`; текст задачи = `--wait --write --model gpt-5.5` + конверт из § 3 SKILL.md.
- **Продолжить тред** — новый вызов с `--resume` в тексте задачи. **Свежий контекст** — `--fresh`.
- **Ворктри** — `--cwd <путь>` в тексте задачи. Не обкатан: при первом параллельном прогоне проверить, что дифф лёг в ворктри, а не в основную копию; до проверки диспатчить последовательно.
- **Финальное ревью** — нативный ревьюер, gh-агент запускает одним Bash-вызовом и возвращает stdout как есть; вердикты триажит оркестратор:

```bash
node "${CODEX_PLUGIN%/}/scripts/codex-companion.mjs" adversarial-review --wait \
  --base <стартовый SHA конвейера> --scope branch --model gpt-5.5 "<оси проверки>"
```

Оси проверки обязательны — это спека ревью; `review` без focus-текста — деградация, не использовать.

### Боевые ограничения

- **Сеть в сандбоксе Codex закрыта** (инцидент 04.07.2026): gh-операции остаются gh-агенту; Codex — только файлы и коммиты.
- **Пустой промпт при спавне** `codex:codex-rescue`: не респавнить — переслать текст задачи тому же агенту (SendMessage), со второго раза доезжает.
